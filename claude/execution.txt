### 3. Protos Path Configuration
**CRITICAL**: Always explicitly set Protos paths to avoid resolution bugs:
```python
import os
from pathlib import Path
from protos.io.paths.path_config import ProtosPaths

# Set environment variables
data_dir = Path("/mnt/c/Users/hidbe/PycharmProjects/MOGRN")
os.environ["PROTOS_DATA_ROOT"] = str(data_dir.absolute())
os.environ["PROTOS_REF_DATA_ROOT"] = str(data_dir.absolute())

# Initialize paths
paths = ProtosPaths(
    user_data_root=str(data_dir.absolute()),
    ref_data_root=str(data_dir.absolute()),
    create_dirs=True
)
```

## � TROUBLESHOOTING

### Path Resolution Issues
If encountering "Path not found" errors:
1. Always use absolute paths
2. Set environment variables explicitly
3. Check ProtosPaths initialization

### Memory Issues
For large datasets:
1. Use cached results when possible
2. Process structures in batches
3. Monitor memory with: `watch -n 1 free -h`

### Missing Dependencies
