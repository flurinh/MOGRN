METHODS

Dataset Assembly and Structure Prediction

The structural dataset comprised 129 microbial rhodopsin structures: 69 experimentally determined structures retrieved from the Protein Data Bank (PDB) and 60 computationally modelled structures generated using Boltz-1. For structure prediction, retinal ligands were specified using a SMILES string representing the beta-ionone ring and polyene chain (excluding the aldehyde oxygen). The covalent Schiff base linkage to the conserved lysine residue was generated automatically during the prediction workflow.

All analyses were restricted to chain A of each structure. Retinal chromophores were identified by residue name (RET, LIG, or LYR) and validated by spatial proximity, requiring at least one heavy atom within 6 A of protein atoms. Functional classifications - proton pump, cation channel, anion channel, chloride pump, sodium pump, sensory rhodopsin, and enzyme regulator - were curated from primary literature. Taxonomic domain assignments (Archaea, Bacteria, Eukaryota, Virus, Synthetic) were likewise annotated manually. Structure identifiers were standardized using a unified naming convention to enable unambiguous one-to-one matching between experimental structures and their corresponding predicted models.

Structure Prediction Validation

Prediction accuracy was assessed by generating Boltz-1 structural models for all 69 experimentally determined structures and quantitatively comparing each model to its experimental counterpart. The paired structures were partitioned into two validation cohorts based on PDB deposition date to account for potential training data overlap. Set A (benchmark set, n = 41) comprised structures deposited before September 2021, the training data cutoff for Boltz-1, and may therefore include sequences or homologous structures present in the model's training corpus. Set B (blind test set, n = 27) contained exclusively post-cutoff structures, providing a rigorous hold-out validation. One structure lacked definitive deposition metadata and was excluded from this temporal analysis. Deposition dates were retrieved programmatically via RCSB PDB metadata queries.

Structural alignments were performed using the CEalign algorithm with default combinatorial extension parameters (window size = 8, maximum gap = 30). To improve alignment robustness for structurally divergent regions, a two-pass refinement procedure was implemented: (i) an initial CEalign pass using all non-retinal C-alpha atoms to establish the global structural correspondence; followed by (ii) a refinement pass restricted to inlier residue pairs - defined as aligned positions with inter-C-alpha distance <=3.0 A after the initial superposition. A minimum of 12 inlier pairs was required to proceed with refinement; otherwise, the initial alignment was retained. To ensure topological consistency, an optional same-helix constraint was enforced, permitting residue pairing only between positions assigned to corresponding transmembrane helices. The rotation matrix and translation vector from the final alignment were applied using QCPSuperimposer for optimal least-squares superposition.

The retinal binding pocket was operationally defined as the set of chain A C-alpha atoms located within 6.0 A of any retinal heavy atom. This distance threshold captures the immediate chromophore environment while excluding peripheral residues not directly involved in ligand coordination.

Prediction accuracy was quantified using three complementary structural metrics:

(1) C-alpha RMSD (backbone RMSD): The root-mean-square deviation of backbone C-alpha atoms after optimal superposition, representing overall fold accuracy across the entire transmembrane domain.

(2) Pocket RMSD: The RMSD of binding pocket C-alpha atoms (residues within 6 A of retinal), calculated using structurally equivalent residue pairs identified through the CEalign path. This metric evaluates local structural agreement in the functionally critical chromophore-binding region.

(3) Ligand RMSD (L-RMSD): The mean closest-atom distance between experimental and predicted retinal heavy atoms after global C-alpha superposition. Unlike conventional atom-pairing RMSD, this metric accommodates potential differences in retinal conformation or protonation state by computing, for each experimental retinal atom, the minimum distance to any predicted retinal atom, then averaging across all atoms.

Structural Conservation Analysis

Transmembrane helix boundaries for the reference structure (CnChR2_J230_refine9) were manually curated through visual inspection of the three-dimensional structure, membrane orientation prediction, and secondary structure analysis. The seven transmembrane helices span the following residue ranges: H1 (84-109), H2 (120-145), H3 (155-176), H4 (188-209), H5 (210-233), H6 (246-271), H7 (281-306). This structure was selected as the global reference based on its central position in the structural similarity network - exhibiting the lowest mean RMSD to all other structures - thereby minimizing cumulative alignment error when propagating annotations across the dataset.

The same reference structure and helix boundaries were used consistently throughout all downstream analyses, including structural alignment, GRN table generation, and distance calculations. Helix boundaries were propagated to all other structures via CEalign residue correspondence tables, which map each reference position to its structurally equivalent position in the target structure.

To further improve alignment accuracy in the retinal-proximal regions most relevant to functional comparisons, a helix-specific distance band filter was applied prior to alignment. For each transmembrane helix, only C-alpha atoms falling within a predefined distance range from retinal were retained for alignment. These bands were empirically determined to capture the core retinal-facing residues: H1 (14.0-17.8 A), H2 (14.0 A), H3 (10.0-11.0 A), H4 (7.5-14.0 A), H5 (9.0-13.0 A), H6 (8.0-17.0 A), H7 (12.0-13.0 A), each expanded by a +/-1.5 A margin. A minimum of 4 residues per helix and 12 residues total were required; otherwise, the filter was bypassed and all C-alpha atoms were used.

Pairwise structural similarities across all 129 structures were quantified as RMSD between transmembrane helix C-alpha atoms (helices 1-7), producing a symmetric 129 x 129 distance matrix. To avoid redundancy, predicted structures with experimentally determined counterparts were excluded from pairwise comparisons, as these would introduce near-identical duplicate entries. Hierarchical clustering was performed on the resulting RMSD matrix using SciPy's implementation of the weighted pair group method with arithmetic mean (WPGMA), which assigns equal weight to each cluster regardless of size, producing balanced dendrograms suitable for visualization.

Five structures (8YEL, HulaCCR1, 4JQ6, AbHeR, 7AKY) clustered as singletons owing to alignment register shifts - systematic offsets of approximately one helical turn (3-4 residues) relative to the reference structure. These shifts produce artificially elevated RMSD values despite preserved overall fold topology. Visual inspection confirmed that these represent CEalign artifacts arising from local sequence insertions or deletions rather than genuine structural divergence. Notably, while structural clustering clearly delineates groups of high pairwise similarity (visible as light-colored blocks along the diagonal), these groupings do not segregate according to molecular function or taxonomic origin, indicating that the seven-transmembrane rhodopsin fold is structurally conserved across diverse functional classes and phylogenetic lineages.

Generic Residue Numbering (GRN) System

The Generic Residue Numbering (GRN) system provides a standardized, structure-based residue indexing scheme for comparative analysis across the structurally diverse microbial rhodopsin superfamily. Unlike sequence-based numbering, which varies with insertions, deletions, and divergent N/C-terminal extensions, GRN assigns positions based on structural equivalence and functional relevance to the retinal chromophore. The system is conceptually inspired by the Ballesteros-Weinstein numbering and GPCRdb nomenclature widely adopted for G protein-coupled receptors, adapted here to the unique heptahelical architecture of microbial rhodopsins. GRN development proceeded through five stages: reference selection, helix boundary definition, structural alignment, anchor determination, and position assignment.

Reference selection. The global reference structure (CnChR2_J230_refine9, a channelrhodopsin-2 variant) was selected based on its lowest mean RMSD to all other structures in the dataset, positioning it at the centroid of the structural similarity network. This central placement minimizes cumulative alignment error when propagating residue annotations transitively across the dataset and ensures optimal alignment quality for structures spanning diverse functional classes.

Helix boundary definition. Transmembrane helix boundaries were manually curated for the reference structure as described above. The same boundaries serve as the coordinate system for all GRN assignments, ensuring consistency between structural alignment and residue numbering.

Structural alignment. All 129 structures were aligned to the global reference using CEalign (window = 8, max_gap = 30), generating residue correspondence tables that map each reference position to its structurally equivalent position in each target structure. Structures exhibiting mean RMSD > 3.0 A to the global reference were flagged for manual inspection and excluded from the final GRN tables, as such large deviations typically indicate alignment artifacts, incomplete structures, or fundamentally different folds rather than genuine structural variation within the rhodopsin superfamily.

Anchor determination. The defining feature universally conserved across all microbial rhodopsins is the retinal chromophore covalently bound to a lysine residue in helix 7. Accordingly, anchor positions were determined by proximity to retinal, ensuring the numbering system is anchored on the functionally conserved binding pocket architecture. For each helix (H1-H6), the anchor position (X.50) was identified as the residue whose side-chain heavy atoms exhibit the minimal mean distance to retinal across all aligned structures. For helix 7, the Schiff base-forming lysine (K296 in CnChR2_J230_refine9) was designated as position 7.50 by definition, given its invariant covalent attachment to the chromophore. This chromophore-centric approach anchors the GRN system on the universally conserved structural and functional element of the rhodopsin superfamily.

Position assignment. Residues were numbered sequentially relative to their helix-specific anchors using the format Helix.Position (e.g., 3.49, 3.50, 3.51). Numbering proceeds from the N-terminal (extracellular) end of each helix toward the C-terminus (cytoplasmic), typically spanning 20-25 positions per helix to capture the full transmembrane segment. Loop regions received systematic identifiers following established conventions: n.XX for N-terminal residues preceding helix 1; c.XX for C-terminal residues following helix 7; and XY.XXX for inter-helix loops, where X and Y denote the flanking helices (e.g., 12.001, 12.002 for the H1-H2 loop; 23.001 for the H2-H3 loop).

Distance profiles. For each GRN position, two complementary distance metrics were computed: (i) side-chain distance - the minimum distance from any side-chain heavy atom to the nearest retinal atom, averaged across all structures; and (ii) C-alpha distance - the distance from the backbone C-alpha atom to the nearest retinal atom, similarly averaged. Side-chain distances capture direct chromophore-residue interactions relevant to spectral tuning and ion selectivity, while C-alpha distances reflect backbone-level spatial relationships indicative of helix packing geometry.

Manual curation. The algorithmically generated GRN tables were subjected to systematic manual inspection and curation to correct alignment artifacts inherent to automated structural comparison. Two principal error classes were identified and corrected:

(1) Helix truncation artifacts: Erroneous gap placement within helices producing artificially shortened transmembrane segments (e.g., residue patterns of the form AA-AA corrected to A-AAAA, where - denotes a gap position). These artifacts arise when CEalign incorrectly fragments a continuous helix into non-contiguous segments.

(2) Register shift errors: Systematic one-turn (approximately 3-4 residue) offsets in helix alignment, manifesting as shifted gap patterns (e.g., AAA---AAAAAA corrected to ---AAAAAAAA). Such shifts arise because the repetitive nature of alpha-helices can yield multiple structurally similar solutions with the helix shifted up or down by one turn, a well-documented limitation of automated structural alignment methods for helical membrane proteins (Stamm & Forrest, 2015).

All corrections were validated by visual inspection of superimposed structures in PyMOL. The curated GRN tables were used for all downstream analyses including distance profile calculations and visualization.

Microbial Rhodopsin Sequence Database Construction

To construct profile Hidden Markov Models (HMMs) for comprehensive rhodopsin sequence identification, we assembled seed alignments from archaeal, eukaryotic, and viral rhodopsin sequences retrieved from NCBI, the UniProt Archive (UniParc), and the Marine Microbial Eukaryote Transcriptome Sequencing Project (MMETSP). This initial search yielded 3,173 sequences, designated subAEV. Bacterial-derived rhodopsin sequences (2,275 sequences; subB) and heliorhodopsin sequences (720 sequences; subH) were collected separately to account for their distinct sequence characteristics.

Profile HMMs for each subfamily (subAEV, subB, subH) were constructed by aligning seed sequences using MAFFT v7.453 with the --genafpair and --maxiterate 1000 options for accurate global alignment. Using these HMMs, rhodopsin genes were extracted from the UniParc database (November 2020 release) and 10,032 metagenomic samples using hmmsearch (HMMER v3.3) with an E-value threshold of 1x10^-5 and a minimum length of 174 amino acids (corresponding to the HMM profile length). This search identified 309,570 candidate rhodopsin sequences. Redundancy reduction using CD-HIT at 90% sequence identity yielded 41,136 non-redundant sequences.

A structure-guided multiple sequence alignment (MSA) was generated from 70 rhodopsin structures with experimentally determined three-dimensional coordinates (Set-A), including 10 channelrhodopsins determined by cryo-electron microscopy (C1C2, bReaChES, CoChR, ChroME2s, CnChR2, KnChR, TsChR, GtACR1, A1ACR1, R2ACR) and 60 additional microbial rhodopsins. Structural alignment was performed using US-align to generate a high-confidence MSA preserving three-dimensional correspondence.

The 41,136 non-redundant sequences (Set-B) were then aligned to the structure-anchored Set-A MSA using MAFFT's --add option for profile-guided alignment. Positions representing rare insertions (gaps in >90% of sequences) were excluded from the final alignment, except where present in Set-A reference sequences. To eliminate contaminant sequences, we identified 134 ungapped positions in the Set-A MSA and removed Set-B sequences containing gaps at 10 or more of these conserved positions, yielding 26,954 high-confidence rhodopsin sequences for downstream analysis.

Protein Similarity Network Analysis

For protein network visualization, we assembled reference rhodopsins (n = 147) with characterized function and/or experimentally determined structure, combined with non-redundant Set-B sequences. Rhodopsin domains were extracted from both reference and Set-B sequences using hmmsearch against the subAEV, subB, and subH profile HMMs. Redundancy within Set-B was reduced using MMseqs2 (v15-6f452) at 60% sequence identity and 50% coverage (--min-seq-id 0.6 -c 0.5 --cluster-mode 2 --cov-mode 0), yielding 5,168 cluster representatives. Combined with reference sequences, this produced a final dataset of 5,315 sequences for network construction.

Pairwise normalized similarity scores were computed using CPAP, which performs all-against-all BLASTp searches (-evalue 1e-2 -dbsize 100000000) and calculates normalized similarity based on high-scoring segment pair (HSP) scores meeting minimum thresholds (>=20% identity, >=20 amino acid alignment length). The normalized similarity score N_AB between proteins A and B is defined as:

N_AB = (1/2)(U_AB + U_BA)

where U_AB represents the position-weighted similarity of protein A to protein B, computed as the sum of maximum length-normalized bit scores at each position of protein A, normalized by self-similarity scores.

The resulting similarity network was visualized using Gephi (v0.10.1), with edges drawn between protein pairs exhibiting normalized similarity scores >=0.2.

Sequence Logo Generation

Rhodopsin sequences from each network cluster were extracted and aligned using Clustal Omega. Sequences lacking complete seven-transmembrane topology or the conserved Schiff base lysine in TM7 were excluded. The filtered alignments were used to generate sequence logos with WebLogo 3.0, representing position-specific amino acid frequencies and information content.

Statistical Analysis and Data Availability

All computational analyses were implemented in Python 3.10. Structural operations including alignment, superposition, and coordinate manipulation utilized Biopython's Bio.PDB module with QCPSuperimposer for optimal rigid-body superposition. Structure file parsing and dataset management employed ProtOS, a custom CIF/mmCIF processing library (Hidber et al., manuscript in preparation). Statistical analyses including Pearson correlation coefficients were computed with two-tailed significance testing. Hierarchical clustering and dendrogram visualization used SciPy's cluster.hierarchy module. All structure coordinates, sequence alignments, RMSD matrices, GRN assignment tables, and analysis code are available at [GitHub repository].
